# Art Round1 Image Two Prompt Pack

## 1. 总体风格约束

所有第一轮资源统一为“架空古典政务策略风”。

统一关键词：

- 架空古典政务、国家治理、内阁会议、政策沙盘、季度奏报。
- 暗金、铜色、旧银、深木、羊皮纸、金属徽章、经济账册、学士院推演。
- 小尺寸 UI 图标清楚可辨，适合 Godot Web 游戏。
- 不现代扁平、不高饱和科技感、不过度写实、不复杂背景、不强书法。
- 优先透明背景 PNG / WebP，边缘干净，适合叠加在深色 UI 上。

通用避免事项：

- 不要大面积纯蓝霓虹、科幻 HUD、现代金融 dashboard 风。
- 不要照片写实人物、不做复杂半身像。
- 不要中文小字写入图片，避免缩小时不可读。
- 不要白底方图；除纹理外都应透明背景。
- 不要过细线条，小尺寸下要保留轮廓。

## 8. 分批生成建议

1. 第一批：6 个角色徽章。用于 DialogueOverlay 体验提升最明显。
2. 第二批：政策类型图标 + 智慧点数图标。用于 PolicyDesk 和政策卡识别。
3. 第三批：锁图标、完成印章、地图四区域图标。用于 LevelSelect 和 MacroMap。
4. 第四批：轻纹理和装饰。确认包体可控后再接。

每批接入后都应检查：

- 中文是否无乱码。
- `index.pck` 是否异常增长。
- Web 首次加载是否明显变慢。
- 缩小到游戏内显示尺寸后是否仍可辨识。
